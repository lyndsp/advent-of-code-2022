# Advent of code 2022 - in python!

Setup based on https://github.com/ollyjshaw/alan_pytridge

## Setup your python environnment

Use `pipenv` !! https://pipenv.readthedocs.io/en/latest/

## Setup the project

`pipenv install --dev`

## Run the tests

`pytest`

## Lint the project

`flake8`

## Run continual testing

`ptw`
